发表于 2020-03-17 | 更新于 2020-03-22 | | | 本文字数 207

    还是记录一下自己的日常，记录自己也是监督自己。今年的寒假很是特别，肺炎的缘故让这个假期格外的长，恰逢自己大学毕业以及研究生考试，惊喜的是自己在2020.02.20查成绩时候发现自己考的比自己预想的还要好一点，339分很有可能能进吉林大学车身工程专硕复试，这一段时间在家的复习效率差到无法形容了，而且自己是跨专业考研，有点浮躁抑郁。一直到最近几天，感觉自己也恢复过来了，着手好好准备复试。不能再浑浑噩噩了。以后每天记录一下自己的心情也会对自己有很大的帮助。
